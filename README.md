# Self-Hosted Talos Cluster with TrueForge ForgeTool

## Overview

This repository documents a self-hosted Talos cluster deployment using:

- Talos Linux for secure Kubernetes nodes
- TrueForge ForgeTool for cluster provisioning and lifecycle management
